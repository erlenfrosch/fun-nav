## Beschreibung
Layout für Smartphone-Nutzung optimieren (Motorrad-Halterung, Handschuhe).

## Aufgaben
- [ ] Große Tap-Targets: Buttons min. 48×48px
- [ ] Bottom Sheet auf Mobile, Sidebar auf Desktop
- [ ] Landscape-Modus: HUD für Querformat optimiert (Karte links, Infos rechts)
- [ ] Wake Lock API: Bildschirm-Schlaf während aktiver Navigation verhindern
- [ ] Overscroll-Verhalten deaktivieren (kein Browser Pull-to-Refresh während Fahrt)

## Akzeptanzkriterium
iPhone/Android: Alle Buttons tappbar mit Handschuhen, HUD in Landscape gut lesbar.
